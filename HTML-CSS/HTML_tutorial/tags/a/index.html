<a> (Anchor)

This tag is used to create Hyper Links that can access other web pages or html pages that are part of the folder hierarchy.

You can set the clickable content of the anchor to be a Text, Image or another HTML content. As long as it is enclosed in the angular brackets of the tag:
the code <a href="https://www.google.com/">Google</a> will generate a link that access the Google.com page:
Google

You can also use it to access other HTML files if you provide the folder path to the file relative to the current HTML file:
the code: <a href="anchor_sub_folder/page02.html">Link to page 2</a> will open the page02.html file:
Link to page 2
